Open source Analog Smartwatch

An open source project to create an hybrid smartwatch that you can build yourself.

Features

  • Activity tracking
  • Count steps, calories burnt, walked distance
  • Sleep trackings
  • Sleep quality: awake, light, deep, duration, overall score
  • Heart rate tracking
  • Tracks your beats per minute, peak and low points
  • Watch info
  • Battery level, change bluetooth name
  • Device Finder
  • Alarm to inform the user of the risk of loss if devices get far from each others, make the smartphone ring to locate it

Components

Specifications proposal, subject to changes

  • Display time
  • Ronda gearbox GB22
  • Microcontroller
  • nRF52832 (nRF52 DK devkit for prototype)
  • Clock
  • DS3231MZ
  • Power Management
  • nPM1100
  • Sleep tracking
  • Accelerometer: BMA400
  • Heart Rate Monitor
  • Optical Sensor: BH1790GLC
  • Watch skeleton
  • Watch Case Butterfly on Aliexpress
  • Battery
  • Round LiPo like LP403535
  • Charger
  • To be defined
